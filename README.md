# オリエンテーリングWebアプリ（仮）

ちずる会向けの、ヒアリング前に触って意見を集めるためのプロトタイプです。ひばりヶ丘駅周辺の実在地点を使った仮コースを収録しています。

地点、導線、所要時間、解説内容は現地調査前の候補です。実施前に安全性と事実関係を確認してください。

## 起動方法

このフォルダで以下を実行します。

```powershell
$env:ADMIN_PASSWORD="当日使う管理者パスワード"
py -3 server.py
```

ブラウザで `http://localhost:8000` を開きます。

## 試せること

- 地図からチェックポイントを選ぶ
- 現在地を地図上に表示し、各地点までの直線距離を確認する
- ヒントと現地ミッションを見る
- GPSチェックイン、またはデモチェックイン
- 発見を文章で記録して得点を獲得する
- 訪問状況と発見ログを見る
- コース説明を見る
- 管理者ページでコース設定とチェックポイントを追加・編集・削除する

コースとチェックポイントは `orienteering.db`（SQLite）に保存され、このサーバーへアクセスする全参加者に共有されます。管理画面の編集にはログインが必要です。

参加者画面は、画面へ戻ったときと30秒ごとに共有データを再取得します。

`ADMIN_PASSWORD` を設定せず起動した場合の試作用パスワードは `chizuru-demo` です。当日運用では必ず別のパスワードを環境変数に設定してください。管理セッションはサーバー再起動時にリセットされます。

参加者の得点と発見ログは、現時点では各参加者のブラウザ内に保存されます。

## GitHub Pages + Supabase

正式な共有データはSupabaseの `orienteering_courses` と `orienteering_checkpoints` に保存します。管理者ログインにはSupabase Authのメールアドレス・パスワード認証を使います。GitHub Pagesはmainブランチへのpush時にActionsから更新されます。

## スマートフォンへ一時公開する

Cloudflare Quick Tunnelを使う方法です。PCとスマートフォンが別のネットワークでもアクセスできます。アカウント登録は不要ですが、PCを起動したままにする必要があり、公開URLは起動ごとに変わります。

```powershell
.\start-public.ps1
```

起動時に10文字以上の管理者パスワードを非表示で入力します。画面に表示される `https://...trycloudflare.com` のURLを参加者へ共有してください。終了するときは `Ctrl+C` を押します。

Quick Tunnelは試作・ヒアリング向けです。当日の常設運用やURL固定が必要になったら、名前付きTunnelまたはクラウドホスティングへ移行します。

同じWi-Fi内のスマートフォンからアクセスさせる場合は、Windowsファイアウォール等を確認したうえで次のように起動します。

```powershell
$env:HOST="0.0.0.0"
$env:ADMIN_PASSWORD="当日使う管理者パスワード"
py -3 server.py
```
