# 土地利用観察ノート

ちずる会のヒアリング結果を反映した、土地利用の違いを観察して記録するための試作アプリです。

チェックポイントはありません。参加者が地図を見て、自分で気づいた場所にピンを置き、名前・視点・コメントを残します。

## 起動

このフォルダで静的サーバーを起動します。

```powershell
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

## 参加者ができること

- 地図をタップして気づきを記録
- 現在地に気づきピンを追加
- 名前、観察の視点、見出し、コメントを保存
- 自分の記録を一覧で見返す
- 共有タブで、だれが何を見てどう気づいたかを一覧する
- 共有タブから全員のピンを地図上で見る

## 管理者ができること

- 観察の視点を追加
- 観察の視点を編集
- 観察の視点を削除
- この端末に保存された記録をリセット

## データの扱い

今の試作では、記録と視点はブラウザの `localStorage` に保存されます。同じ端末・同じブラウザ内で共有ボードとして見られる形です。

複数端末の参加者が同時に記録を共有する本番運用にする場合は、Supabase などの共有データベースに保存先を切り替える必要があります。
